# 🛡️ Cyber Scam Detection Dashboard

🔐 A cyber security web tool that detects **suspicious URLs and scam messages**.
This project analyzes links and messages to identify possible **phishing or scam attempts** and shows the results in a **cyber security dashboard**.

🚀 Features
- 🔎 URL phishing detection
- 💬 Scam message detection
- 📊 Threat analysis dashboard
- 📈 Risk percentage chart
- 🕘 Scan history graph
- 📄 Security report PDF download
- 📱 Responsive UI (mobile, tablet, desktop)

 🛠️ Tech Stack

- 🐍 Python
- 🌐 Flask
- 🎨 HTML
- 🎨 CSS
- ⚡ JavaScript
- 📊 Chart.js
- ✨ Anime.js
- 📄 jsPDF

⚙️ Installation

Clone the repository
git clone https://github.com/satyamsahni27-debug/cyber-scam-detector
Go to the project folder

cd cyber-scam-detector

Install dependencies

pip install -r requirements.txt

Run the application

python app.py

 🖥️ Usage

1️⃣ Enter a website URL 🌐  
2️⃣ Paste a suspicious message 💬  
3️⃣ Click the scan button 🔍  
4️⃣ The dashboard will display risk analysis and threat score 📊
